Pattern Libraries

We often develop our complate components using a pattern library. The following pattern libraries support complate:

  • Fractal (Sample project)
  • aiur

Adapters for Server-Side Rendering

With complate we have adapters so that we can reuse our components for server-side rendering across multiple platforms.

The following platforms currently support complate:

  • Java and an adapter for the Spring Framework (Sample project)
  • complate works natively with JavaScript and we have an adapter for the Express Framework (Sample project)
  • Ruby on Rails (Sample project)
