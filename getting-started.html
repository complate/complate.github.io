Pattern Libraries

We often develop our complate components using a pattern library. The following pattern libraries support complate:

  • Fractal (sample project)
  • aiur

Adaptors for Server-Side Rendering

With complate we have adaptors so that we can reuse our components for server-side rendering across multiple platforms.

The following platforms currently support complate:

  • Java and an adaptor for the Spring Framework (sample project)
  • complate works natively with JavaScript and we have an adaptor for the Express Framework (sample project)
  • Ruby on Rails (sample project)
